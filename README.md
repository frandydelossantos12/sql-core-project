# sql-core-project

Este proyecto contiene la base de datos relacional central para un sistema de e-commerce simulado.

## Descripción
La base de datos está diseñada como un sistema transaccional (OLTP) que almacena
información de ventas y sirve como base para procesos de ETL y análisis en Power BI.

## Características
- Modelo relacional normalizado
- Preparado para grandes volúmenes de datos
- Diseñado para consumo analítico y BI

## Entidades principales
- Clientes
- Empleados
- Productos
- Marcas
- Ventas
- Dimensión de fechas

## Contexto de portafolio
Esta base de datos forma parte de un portafolio de datos que incluye:
- Un pipeline ETL para limpieza y transformación
- Un dashboard en Power BI para análisis ejecutivo
